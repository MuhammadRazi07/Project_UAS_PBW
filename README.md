# Z Aksesoris

Z Aksesoris adalah aplikasi web marketplace berbasis Laravel yang menyediakan layanan penjualan pakaian anak, tas anak, aksesoris, dan mainan edukatif secara online.

## Fitur Utama

### User

* Registrasi dan Login
* Melihat daftar produk
* Mencari produk berdasarkan nama
* Melihat detail produk
* Menambahkan produk ke keranjang
* Menambahkan produk ke wishlist
* Checkout pesanan
* Melihat riwayat pesanan
* Mengelola profil akun

### Admin

* Login admin
* Kelola produk (Tambah, Edit, Hapus)
* Melihat seluruh pesanan pelanggan
* Mengubah status pesanan:

  * Pending
  * Diproses
  * Dikirim
  * Selesai
* Dashboard administrasi

## Teknologi yang Digunakan

* Laravel 12
* React.js
* Inertia.js
* Tailwind CSS
* MySQL
* Vite

## Tujuan Aplikasi

Aplikasi ini dibuat sebagai proyek UAS Pemrograman Berbasis Web dengan tujuan menerapkan konsep CRUD, autentikasi, relasi database, middleware, serta integrasi Laravel dan React dalam pengembangan aplikasi modern.
